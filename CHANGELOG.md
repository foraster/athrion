# Changelog

## [2.0.0] - 2025-06-15
### Changed (Server)
- Added `isActive` and `stockQuantity` fields to Product model.
- Added `shippingAddress` and `paymentMethod` to Order model.
- Controllers updated to handle new fields.

## [0.2.0] - 2025-06-15
### Changed (Client)
- Admin dashboard improved with CSV import/export and metrics.
- Product table reworked.
- Split `ProductForm` into multiple components.
- Added admin settings components.
- Improved all admin-related components.

**Note:** This client version requires **server >= 2.0.0**
