# Athrion

Athrion is a full-stack e-commerce platform for a modern fitness equipment brand built with React, Express, and PostgreSQL. It includes a customer storefront and an admin dashboard for catalog management, order handling, and basic sales analytics.

The project focuses on full-stack architecture, authentication, business workflows, and a structured client/server setup.

## Tech Stack

Frontend: React, React Router, MobX, Axios, Bootstrap, Tailwind CSS, Recharts  
Backend: Node.js, Express, Sequelize, PostgreSQL, JWT, bcrypt, express-fileupload

## Key Features

- Customer storefront with catalog browsing, filtering, and checkout flow
- User profile with order history
- Admin dashboard for product and order management
- CSV import/export and basic sales analytics
- Secure authentication and role-based access control

## Getting Started

To run Athrion locally:

1. Clone the repository
2. Set up environment variables in `server/.env` (see `.env.example`)
3. Install dependencies in both `client/` and `server/`
4. Start both servers

Requirements: Node.js, PostgreSQL
